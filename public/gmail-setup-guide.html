Update .env.local File

Add the App Password to your project configuration.

  1. Open project folder in VS Code: D:\amit_sir_website
  2. Find and open file: .env.local
  3. Find this line:
    GMAIL_APP_PASSWORD=your-app-password-here
  4. Replace with your password (remove spaces):
    GMAIL_APP_PASSWORD=abcdefghijklmnop
  5. Save the file (Ctrl+S)
‚ö†Ô∏è Remove all spaces from the password!
‚ùå Wrong: abcd efgh ijkl mnop
